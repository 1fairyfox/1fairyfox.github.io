### The theme was not planned, and it is a good one

Two very different projects spent the same day on the same discipline: **hand your work to something
outside yourself and let it grade you.**

**Pokered Save Editor 2** had its biggest day so far. The map screen stopped drawing a picture and
started running the game's own overworld pipeline (`0.17.0`); a legally backed-up cartridge, booted
headless in PyBoy, became the judge (`0.18.0`). It said **no** about the border ring — held as an
explicit `QEXPECT_FAIL`, not a skip — and connection strips closed it (`0.19.0`, 78/78 verified against
the ROM's own compiled structs). The six glitch palettes turned out to be a *pointer subtraction*, so
exactly four contrast levels and exactly six glitch palettes exist, all ten confirmed on the console
(`0.20.0`). Then a **DMG APU + Gen 1's sequencer in C++**, graded by the cartridge's own 256 bytes of
engine state photographed every frame — 46 tracks byte for byte (`0.22.x`). That parity test caught
three bugs of *tidiness* (the port had done the right thing where the game does the wrong one) and
caught the **oracle itself** being wrong. And the "glitch music" turned out to be 105 real inner voices,
because song ids are computed from header addresses. The meaning layer (`0.24.0`) then uncovered a
**save-corrupting bug** that had been shipping for years — shared collision lists, an importer one step
out of phase, and the editor *writes* that pointer, so a Poké Mart wrote a bedroom's walls.

**Random AI Prompt** did the same thing from the other side: it put the Android build on a **real
Android runtime** and measured the 1,000-prompt promise it had been asserting (`2.60.0`) — 253 ms to
render, memory flat as the list grows 50×. The "defect" the gate first reported was **the test's own**,
and the instrumentation added to diagnose it is what disproved it. Then a **critical command injection**
CodeQL found on the release PR (`2.60.1`): a filename could close the quote in a shell string and run the
rest. Fixed structurally — no shell at all, program and argv handed to `execFile`.

**Fairy Fox Games** planted **Brim** (`0.22.0`), the 13th game and a new verb (pour), whose hook is a
delay line — the stream keeps arriving after you close the spout. **Fairy Fox Stories** ran a pure grow
day (`0.3.4`) and *held* its plant because its own cadence gate said it wasn't due.
